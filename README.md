# TestSeleniumJavascript
Selenium WebDriver using Javascript, mocha and chai and chai as promised

# How to install
`git clone git@github.com:salsakiuno/testSeleniumJavaScriptCMP.git`

# Install libraries
`npm install`

# How to run
`npm test`
